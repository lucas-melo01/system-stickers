## 🔴 PROBLEMAS CRÍTICOS ENCONTRADOS E CORRIGIDOS

### 1. **DbUpdateException: NOT NULL violation on `Pedidos.Id`**
- **Causa**: Migrations SQLite vs Runtime Postgres (Identity não gerado)
- **Solução**: Migration `MigrateToPostgres` com `GENERATED BY DEFAULT AS IDENTITY`
- **Status**: ✅ RESOLVIDO

### 2. **PostgresException 42804: DateTime TEXT → timestamp with time zone**
- **Causa**: `DataPedido` e `DataCriacao` como TEXT no Postgres
- **Solução**: Cast seguro com `USING "DataPedido"::timestamp with time zone`
- **Status**: ✅ RESOLVIDO

### 3. **PostgresException 42804: Integer boolean → boolean**
- **Causa**: `Impresso` e `Erro` como INTEGER no SQLite
- **Solução**: `DROP DEFAULT` → cast → `SET DEFAULT false`
- **Status**: ✅ RESOLVIDO

### 4. **Relacionamentos EF não configurados explicitamente**
- **Causa**: Dependência de convenções EF
- **Solução**: `OnModelCreating` com configuração explícita
- **Status**: ✅ RESOLVIDO
